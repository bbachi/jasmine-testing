# jasmine-testing
Basic standalone jasmine testing

## To edit tasks file in vscode to run directly from vscode

### To open tasks file

`cntrl + shift + p`

### replace this

`
  {
		"version": "0.1.0",
		"command": "explorer",    
		"windows": {
			"command": "explorer.exe"
		},
		"args": ["${file}"]
	}
`

### To run directly from VSCode

` cntrl + shift + b`
